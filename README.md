# Treenity Experimental


#### Based on
* React
* Mobx-State-Tree
* Feathers JS
* Parcel Builder

#### PreRequesites
* Node 12+
* Yarn
* MongoDB Server

#### Installation
1. `yarn` to install dependencies
1. `yarn serve-client`
1. `yarn watch-server` in spare terminal window
1. `node dist/index.js` to run server

Open <http://localhost:3101> in browser
